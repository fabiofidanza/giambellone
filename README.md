	giambellone.renderGauges('.gauge',
		{
			thickness: 0.2,
			animate: true,
			foregroundColor: "#EC008C",
			backgroundColor: "#eee",
			animateDuration: 2000,
			animateEasingExponent: 2
		});

Pass a selector, an Element or an Array of Element(s).
Pass your options or let it use decent defaults.
